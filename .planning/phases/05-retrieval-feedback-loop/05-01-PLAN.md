**Step 4: Create src/retrieval/orchestrator.ts (per D-01, D-03, D-04)**

Export interface:
```typescript
export interface CoverageResult {
  covered: boolean;
  articles: Article[];
}
```

Export `assessCoverage(question: string, store: WikiStore, threshold: number): Promise<CoverageResult>`:
- Import `buildIndex`, `search` from `../search/search-index.js`
- Import `WikiStore` type from `../store/wiki-store.js`
- Import `Article` type from `../types/article.js`
- Guard: `const articles = await store.listArticles(); if (articles.length === 0) return { covered: false, articles: [] };`
- Build index: `const index = buildIndex(articles);`
- Search: `const results = search(index, question);`
- Check top: `const topResult = results[0]; if (!topResult || topResult.score < threshold) return { covered: false, articles: [] };`
- Collect context articles (top 5): `const contextSlugs = results.slice(0, 5).map(r => r.slug);`
- Load full articles: `const contextArticles = (await Promise.all(contextSlugs.map(slug => store.getArticle(slug)))).filter((a): a is Article => a !== null);`
- Return `{ covered: true, articles: contextArticles }`
